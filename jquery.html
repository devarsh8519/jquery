• Q1 :- What is jQuery?

  • Solution :-

    jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers.

    • Features of jquery :

  • Extensibility through plug-ins
  • DOM manipulation
  • JSON Parsing
  • CSS manipulation
  • HTML manipulation
  • Utilities
  • AJAX
  • DOM element selection
  • HTML event methods
  • Effects and Animations

  • Q2 :- How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery ,JQuery Animation ?

  • Solution :-

    • How to Apply CSS Using JQuery :-

  • CSS will be appiled by $ symbol

  • Syntax : - css("propertyname");

  • Example 1 : - $("p").css("background-color");

  • Example 2 : - $("p").css({"background-color": "green", "font-size": "100%"});

    • Add and Remove Class in Jquery

    Add Class :-

  • Syntax:- $('selector').addClass(class_name);

  • Example:- $('#btnAdd').click(function () { $('h1').addClass("heading box") })

    • Remove Class :-

  • Syntax:- $('selector').removeClass(class_name);

  • Example:- $('#btnRemove').click(function () { $('h1').removeClass("heading box") })
